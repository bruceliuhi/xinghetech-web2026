```html
<!DOCTYPE html>

        const images = [

            'hero-background.png', 'agent-background.png',

            'hero-visual.png', 'decorative-pattern.png', 'og-image.jpg'
        ];
        
        const container = document.getElementById('images');
        images.forEach(src => {
            const div = document.createElement('div');
            div.style.display = 'inline-block';
            div.innerHTML = `
                <div><strong>${src}</strong></div>
                <img src="/images/${src}" 
                     onerror="this.parentElement.classList.add('missing'); this.style.display='none'; this.nextSibling.style.display='block';"
                     onload="this.nextSibling.style.display='none';">
                <div style="display:none" class="missing">❌ 缺失</div>
            `;
            container.appendChild(div);
        });
    </script>
</body>
</html>
```

## 🆘 常见问题

### Q1: 某些图片不显示

**排查步骤：**

1. 检查文件名是否完全匹配（区分大小写）
   ```bash
   ls public/images/
   ```

2. 检查文件格式是否正确
   - Logo 和 QR 必须是 SVG
   - 其他可以是 PNG/JPG

3. 检查文件是否损坏
   - 尝试在图片查看器中打开

### Q2: 图片显示但很模糊

**解决方案：**
- 使用更高分辨率的图片
- 确保图片尺寸符合建议值
- 使用 PNG 代替 JPG（保留透明度）

### Q3: 图片加载很慢

**优化方法：**

```bash
# 安装图片压缩工具
npm install -g imagemin-cli

# 压缩图片
imagemin public/images/*.png --out-dir=public/images/optimized
```

### Q4: 想批量替换图片

使用脚本批量重命名：

```bash
# 示例：将 background-1.png 重命名为 hero-background.png
mv public/images/background-1.png public/images/hero-background.png
```
