## 方式一：自动下载占位图（最快 - 推荐测试用）⚡

### 3 步完成

```bash
# 第 1 步：安装依赖
npm install

# 第 2 步：自动下载所有占位图
npm run download-images

# 第 3 步：启动项目
npm run dev
```

**完成！** 访问 http://localhost:5173，所有图片都会显示。

- ✅ 自动下载 23 个占位图片
- ✅ 使用品牌色 (#2C4BFF、#9122FA)

1. 启动项目：`npm run dev`
2. 打开浏览器：http://localhost:5173
3. 按 F12 打开开发者工具
4. 切换到 **Network** 标签
5. 筛选 **Img** 类型
6. 刷新页面

## 🆘 常见问题

**Q: 自动下载失败怎么办？**
- 检查网络连接
- 尝试方式三手动下载
- 或准备真实设计图

**Q: 某几张图片不显示？**
- 运行 `npm run check-images` 查看缺失哪些
- 检查文件名是否完全匹配（区分大小写）
- 确认图片放在 `public/images/` 目录

**Q: 图片很模糊？**
- 占位图本身清晰度有限
- 建议使用真实设计图
