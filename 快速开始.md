# ⚡ 快速开始指南

## 下载代码后的 3 步部署

### 步骤 1: 安装依赖 (1分钟)

```bash
cd 星河卓越AI平台
npm install
```

### 步骤 2: 检查图片 (30秒)

```bash
npm run setup
```

查看输出报告，了解哪些图片需要准备。

### 步骤 3: 启动项目 (10秒)

```bash
npm run dev
```

访问 http://localhost:5173 ✅

## 图片处理方案

### 🎯 方案一：先运行，后完善（推荐新手）

**直接启动，使用已有的占位图**

```bash
npm install
npm run dev
```

项目已包含 3 个关键 SVG 占位图：
- ✅ Logo
- ✅ 二维码  
- ✅ 实践背景

其他缺失图片会显示为纯色/渐变背景，**不影响基本浏览**。

之后根据需要逐步替换图片。

---

### 🚀 方案二：快速测试（使用临时图片）

```bash
npm install
npm run setup
bash scripts/download-placeholder-images.sh  # 下载临时占位图
npm run dev
```

⚠️ 注意：这些是从 Unsplash 下载的临时图片，仅用于测试。

---

### ⭐ 方案三：正式部署（使用真实设计图）

1. **安装依赖**
   ```bash
   npm install
   ```

2. **查看图片清单**
   ```bash
   npm run setup
   cat 图片清单.txt
   ```

3. **准备图片**
   
   将设计图片放入 `public/images/` 目录：
   
   ```
   public/images/
   ├── logo.svg                    ✅ 已提供
   ├── qr-code.svg                 ✅ 已提供
   ├── hero-background.png         ← 准备这个
   ├── agent-background.png        ← 准备这个
   ├── 3d-chat-icon.png           ← 准备这个
   └── ... (其他图片)
   ```

4. **验证图片**
   ```bash
   npm run check-images
   ```

5. **启动项目**
   ```bash
   npm run dev
   ```

## 🆘 遇到问题？

### ❌ 安装依赖失败

```bash
# 清除缓存重试
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### ❌ 端口被占用

修改端口（编辑 `vite.config.ts`）：

```js
export default {
  server: { port: 3000 }  // 改为其他端口
}
```

### ❌ 页面白屏

1. 检查 Node.js 版本 >= 16
2. 查看浏览器控制台错误
3. 清除浏览器缓存

### ❌ 图片不显示

正常！缺失的图片会用纯色代替，不影响功能。

需要时运行：
```bash
npm run check-images  # 查看缺失哪些图片
```
