## 🔍 定位某个图片的位置

### 方法1：按文件名找
```
打开项目 → public/images/ → 找到对应文件
```

### 方法2：按页面找
```
首页         → hero-background + 6个3D图标
平台能力页   → 2个背景 + 6个插图
Agent页      → 1个主视觉 + 2个背景
实践页       → 1个背景
导航栏/页脚  → Logo + 二维码
```

### 方法3：按功能找
```
导航栏 Logo   → logo.svg
页脚 Logo     → logo.svg
页脚二维码    → qr-code.svg
浏览器图标    → favicon.ico
```
