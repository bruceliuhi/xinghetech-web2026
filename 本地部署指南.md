## 🚀 快速开始

### 1. 下载并解压代码
```bash
# 解压下载的代码包到任意目录
cd 星河卓越AI平台
```

### 2. 安装依赖
```bash
# 使用 npm
npm install

# 或使用 yarn
yarn install
```

### 方案二：使用 Unsplash 占位图片（快速测试）

如果暂时没有设计图片，可以使用以下命令生成占位图片链接：

```javascript
// 在浏览器控制台运行，自动下载测试图片
const images = [
  { name: 'hero-background.png', search: 'technology abstract' },
  { name: 'agent-background.png', search: 'ai neural network' },
  { name: '3d-chat-icon.png', search: 'chat bubble 3d' },
  // ... 更多
];

images.forEach(img => {
  console.log(`${img.name}: https://source.unsplash.com/800x600/?${img.search}`);
});
```

使用在线工具生成纯色或渐变占位图：
- [PlaceHolder.com](https://placeholder.com/)
- [DummyImage](https://dummyimage.com/)

示例：
```
https://via.placeholder.com/1920x1080/2C4BFF/FFFFFF?text=Hero+Background
```

## ❓ 常见问题

### Q1: 图片不显示怎么办？

**检查清单：**
1. ✅ 图片文件是否存在于 `public/images/` 目录
2. ✅ 文件名是否完全匹配（区分大小写）
3. ✅ 文件格式是否正确（.png, .jpg, .svg）
4. ✅ 图片文件是否损坏

### Q2: 安装依赖失败

```bash
# 清除缓存后重试
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### Q3: 端口被占用

修改 `vite.config.ts` 中的端口：

```typescript
export default defineConfig({
  server: {
    port: 3000, // 改为其他端口
  },
});
```

### Q4: 页面白屏或报错

1. 检查浏览器控制台错误信息
2. 确保 Node.js 版本符合要求
3. 尝试清除浏览器缓存
4. 重新安装依赖

### Q5: 图片加载很慢

**优化建议：**
- 压缩图片大小（推荐使用 TinyPNG）
- 使用 WebP 格式
- 为大图添加懒加载
- 使用 CDN 托管图片

## 🎨 自定义配置

### 修改品牌色

编辑 `src/styles/theme.css`：

```css
:root {
  --primary: #2C4BFF;      /* 主色 */
  --accent: #9122FA;       /* 强调色 */
  --background: #FFFFFF;   /* 背景色 */
}
```

### 修改 Logo

替换以下文件：
- `public/images/logo.svg` - 导航栏 Logo
- `public/favicon.ico` - 浏览器图标
